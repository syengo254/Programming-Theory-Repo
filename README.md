# Programming-Theory-Repo
Submission: Programming theory in action

## Game Name
Avoid The Blocks!

Simple 3D Game using primitive shapes.

## Instructions
1. Clone the repo
2. Open project in Unity 2022 LTS
3. Run to Play - Use LEFT/RIGHT Arrow Keys to move the player
